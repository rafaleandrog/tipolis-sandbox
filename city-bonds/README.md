# city-bonds

## Purpose
Project workspace for city bonds.

## Structure
- index.html: Entry page for GitHub Pages.
- pages/: Standalone content pages.
- assets/: Project CSS, JS, images, documents, and uploads.
- apps/: Web apps and tools.
- dashboards/: Monitoring and analytics views.
- research/: Research material and references.
- data/: Datasets and models.
- components/: Reusable UI/components.
- experiments/: Experimental work.
- archive/: Deprecated/inactive assets.

## Deployment Notes
Use relative links and keep static hosting compatibility.
